# Code for: paper titled 'Digital Exclusion and Financial Inclusion in Later Life: Stable and Unstable Variables of Bank Account Ownership'.

## Description
This repository contains code and analysis for the research of financial inclusion and digital exclusion of older adults, using tree-based models, XAI techniques, to analyze the consistency of variables' rankings.

## Environment
Create environment using:
conda env create -f environment.yml

## Run
Open:
financial_inclusion_tree_models_XAI_kendalW_clean.ipynb

and run all cells.

## Note
Repository is anonymized for blind review.
